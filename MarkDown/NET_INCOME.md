# Net Income Feature

## Overview
Ang sistema ay mayroon nang Net Income display sa Revenue & Expenses tab ng project view.

## Location
- **Path**: Projects → Description → View Project → Revenue & Expenses Tab
- Ang Net Income ay automatically calculated at ipinapakita sa tab na ito

## Calculation
Ang Net Income ay calculated as:
```
Net Income = Total Revenue - Total Expenses
```

## Display
- Ang Net Income ay ipinapakita kasama ng Revenue at Expenses summary
- Parehong format ang ginagamit tulad ng nasa Revenue & Expenses Module Page
- Real-time ang update ng Net Income kapag may changes sa revenue o expenses
