# Close/Lock Project Feature

## Overview
Ang sistema ay mayroon nang feature na pwedeng i-lock o i-close ang isang project para maiwasan ang mga accidental changes.

## Features

### 1. Project Locking
- Pwedeng i-lock ang isang project para hindi na ito ma-edit
- Kapag naka-lock na ang project, kahit ang Master Admin ay hindi na pwedeng mag-edit
- Ang project ay magiging view-only mode

### 2. Locked Project Restrictions
Kapag naka-lock na ang project, hindi na pwedeng:
- Mag-add ng bagong revenue entries
- Mag-add ng bagong expense entries
- Mag-add ng bagong billing entries
- Mag-add ng bagong collection entries
- Mag-delete ng existing entries
- Mag-edit ng existing entries

### 3. Permanent Lock
- Ang locked project ay hindi na pwedeng i-unlock
- Ito ay permanent action para mapanatili ang data integrity

## How to Use
- Ang feature na ito ay available para sa System Administrator at Project Manager roles
- Ang Master Admin ay mayroon ding access sa feature na ito
